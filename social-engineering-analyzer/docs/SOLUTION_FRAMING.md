## User Story Template

**As a** [specific user type],  
**I want to** [specific action],  
**So that** [specific benefit].

### Example:
**As a** crypto user receiving suspicious DMs,  
**I want to** quickly check if a message is a scam,  
**So that** I don't lose my funds to fraudsters.

- [ ] Feasible MVP scope (can build in timeframe)

- [ ] Plan for validation/testing
